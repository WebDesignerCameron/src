# HTML=>HTML
## Summary
Injecting a HTML page's content into
a container. 
## Using
Connect this with `<script src="https
://github.com/WebDesignerCameron/src/
templates/js/implementation/
HTML_into_HTML/implement.js"></script>
`, and make sure the file you are
implementing is called `content.html`,
and add `<div id="external-
container">`.
