# All
## Summary
Spreadsheets incorporating the main, 
footer and header elements all at 
once. 
## Using
### Colour
`col.css` sets the header and footer 
to a dark theme and the main to a 
light theme. `invert_col.css` does 
the opposite. `dark.css` makes it all 
dark, `light.css` makes it all light. 
### Margin
`marg_number_hf.css` gives the 
header a margin-bottom of `
number-px` and the footer a margin-top 
of `number-px`.
