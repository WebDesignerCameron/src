# Hero
## Summary
The `class="hero"` section, also known 
as the introduction of the page. 
## Using
`glowing.css` makes a bright gradient 
in the hero section. `article.css` 
is for the articles, buttons or 
divs inside the hero section, and 
gives them a nice pastel `#ff6b6b` 
colour, nicely contrasting with 
`glowing.css`'s gradient.
