# Reset
## Summary
Global resets including margin, 
padding, and box sizing. 
## Using
Each name corresponds to a certain 
reset. The resets apply to all 
elements. `p` means a `padding: 0;`  
reset, `m` means a `margin: 0;` 
reset, and `b` means a `
box-sizing: border-box;` reset. 
`reset.css` resets all three.
